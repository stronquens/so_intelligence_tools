# Sync Session Documentation

## Why

The Windows implementation now includes more than selected-text correction: the main overlay opens with `Ctrl + Alt + A`, push-to-talk dictation works on Windows with a warm faster-whisper Docker backend, and the Electron overlay/settings bridge has real behavior for settings and selected-text correction.

Some high-level documentation still described Windows as selected-text-only or listed the overlay as future work.

## What Changes

- Update README status, Windows quick start, roadmap and limitations.
- Update AGENTS project context with the current operational Windows state.
- Update docs index and architecture overview.
- Update Windows support and desktop UI docs with the overlay shortcut and current bridge behavior.

## Scope

Documentation only. No runtime behavior changes.
