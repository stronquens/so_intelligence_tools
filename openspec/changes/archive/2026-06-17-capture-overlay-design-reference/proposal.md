# Capture Overlay Design Reference

## Problem

The project now has a concrete visual direction for the future system overlay, but the reference image only exists as a transient Codex attachment. It should be preserved in the repository so future overlay work can align with the intended product feel.

## Scope

- Store the provided overlay mockup as a versioned design asset.
- Link the asset from desktop UI documentation.
- Add a tools-overlay requirement that future overlay implementation should use this reference as visual guidance.

## Non-Goals

- Implement or restyle the overlay UI now.
- Change keyboard shortcuts or runtime behavior.
- Treat the reference image as a pixel-perfect specification.
