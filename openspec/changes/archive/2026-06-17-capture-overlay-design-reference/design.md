# Design

## Decision

Store the reference image at:

```text
assets/design/overlay-future-reference.jpg
```

This keeps product design references separate from runtime screenshots in `assets/screenshots/`.

## Visual Direction

Future overlay iterations should use the image as guidance for:

- translucent glass-style panels
- compact launcher grid for primary tools
- visible settings access from the overlay
- shortcut editing surface beside or near the launcher
- calm desktop-native styling rather than a marketing page

The image is a product direction reference, not a strict pixel-perfect contract.

## Risks

- The mockup may include Linux-like desktop chrome while Windows support is now active. Future implementation should preserve the product feel while adapting window behavior and controls to each platform.
